1. フォームの作成

◆ タグのおさらい

これまでに、HTML フォームとパーツを構成するタグを学習しました。
復習も兼ねて、本章では各フォームを作成しましょう。


●input タグ

フォームを構成するパーツを設定します。
基本的なコードは以下の様に記述し、閉じタグは不要です。

<input type="種類" />

パーツの種類は、type 属性で指定します。以下はその一例です。

  • text ：テキストボックスを作成
  • password：入力値を非常時にする
  • date：日付の入力
  • radio：ラジオボタン
  • checkbox：チェックボックス
  • file：ファイルのアップロード

●label タグ

フォームの各パーツに対してラベル(名前)を付けます。
ラベルは開始タグと終了タグの間に記述します。また、
パーツのタグを囲む記述方法と、for 属性を指定する記述方法があります。

<!--タグで囲む-->
<label>ラベル<input type="種類" /></label>

<!--for属性で指定する-->
<label for="値">ラベル</label>
<input type="種類" id="値" />

●textarea タグ

入力文字を指定できるテキストボックスを作成します。
cols 属性で列数、rows 属性で行数を指定できます。

<textarea cols="列数" rows="行数">デフォルト文</textarea>

●select タグ

select タグで option タグを囲い、option タグのコンテンツに
選択肢を記述する事で、プルダウンメニューを作成します。
属性は主に 3 種類あり、select タグのsize 属性では
選択肢の表示行数を指定します。
select タグのname 属性と、option タグの value 属性は
同じ値を入れる必要があります。

<select name="値" size="表示桁数">
  <option value="値">リスト1</option>
  <option value="値">リスト2</option>
  <option value="値">リスト3</option>
</select>

ボタンは button タグを使います。以下 3 つの属性を指定できます。

  • submit：フォーム等の情報を送信する
  • reset：フォームの内容を削除する
  • button：ボタンを作成する

2. 疑似要素と疑似クラス

◆ 疑似要素とは

HTML の要素を、疑似的に CSS で設定する記述方法です。
セレクタにキーワードを追加し、要素の場所に紐付けて指定します。
各 疑似要素に対し、一部のプロパティが利用可能です。


● 基本構造

疑似要素を反映させたいセレクタの後ろに、
:: (コロン 2 つ)を記述し、キーワードを続けます。

セレクタ::キーワード {
  プロパティ: 値;
}

ここからは、具体的な疑似要素を学習して行きましょう。



●before と after

before は指定した要素の最初、after は指定した要素の最後に
要素を追加します。contentプロパティを記述し、
フォントや背景に関する各プロパティを設定します。

[html]

<p class="rank1">〇〇ランキング 第1位</p>
<p class="rank2">〇〇ランキング 第2位</p>
<p class="rank3">〇〇ランキング 第3位</p>

[CSS]

p::before {
  content: "»";
  color: blue;
}

.rank1::after {
  content: " ← おススメ！";
  color: red;
  font-size: 13px;
  background-color: lightgreen;
}
演習 1
上記のコードを記述し、結果を確認しましょう。
また、ランキング 2位に対し、after疑似要素で任意のプロパティを追加しましょう。

疑似要素は CSS 上で編集が可能なため、注釈文や画像を試す時に用いると便利です。 その他の例を確認していきましょう。


●first-letter と first-line

first-letter は要素の文頭に、first-line は要素の一行目にスタイルを適用させる疑似要素です。

[html]

<p class="sample1">本日のニュースです。</p>
<p class="sample2">
  東京事業所でプログラミング勉強会が開催されました。<br />
  今回のテーマは、疑似要素と疑似クラスです。
</p>

[CSS]

.sample1::first-letter {
  color: tomato;
  font-size: 150%;
}
.sample2::first-line {
  color: green;
  font-size: 120%;
}
演習 2
上記のコードを記述し、結果を確認しましょう。

●selection

対象のコンテンツに対して、選択した際の変化を指定します。
以下を例とする、特定のプロパティのみ利用可能です。

・color
・background-color
・text-shadow

[html]

<p class="sample1">選択すると文字色と背景が変化します</p>
<p class="sample2">選択すると影が追加されます。</p>

[CSS]

.sample1::selection {
  color: red;
  background-color: lightgreen;
}

.sample2::selection {
  text-shadow: 2px 2px 3px skyblue;
}

◆ 疑似クラス

選択した要素に対し、状態に基づいてスタイル付けします。
要素の状態とは「マウスカーソルが接触した」「クリック時」等を指します。


● 基本構造

疑似クラスを指定したい要素のセレクタの後ろに、
: (コロン 1 つ)を記述し、キーワードを続けます。

●hover

マウスポインタが、要素の上を通過した際のスタイルを指定します。
※ hover は通過の意味

[html]

<button>Change blue</button>

[CSS]

button:hover {
  color: blue;
}

●active

リンク等をクリックした時のスタイルを指定します。

[html]

<a href="#">このリンクをクリックしてください。</a>

[CSS]

a:active {
  color: red;
}

3. flexbox(1)

◆flexbox とは

CSS3 から利用できる ボックスモデルのレイアウトです。
主軸と交差軸が設定され、その軸に対してコンテンツの位置関係を指定します。

● 基本構造

まずflex-container ボックスがあり、
その中にflex-item ボックスが配置されます。

この時、flex-container に対して軸が 2 本設定され、
flex-item の並べ方や位置を指定する事が出来ます。



◆flex container と flex-item

●flex-direction

display:flex を追記すると、10 個のボックスが
並び替えられました。これは<div class="container">
配下の子要素が flex-item に指定されたためです。
(対象は直下の子要素だけ)

ここからは、flex-item の並べ方を学習していきます。
flex-directionは、flex-item の並び方向を指定するプロパティで、
デフォルトは main axis の方向(今回は横並びの row)になっています。
この値を変更する事で、flex-item の並べ方を指定できます。


演習 2
下記のコードを参照し、2つの flex-direction それぞれの結果を表示させましょう。
※ flex-direction の Emmetは fxd, fxd:c, fxd:cr 
.container {
display: flex;
flex-direction: column; /* 縦並び */
flex-direction: column-reverse; /* 縦反転並び */

●flex-wrap

flex-wrap は、flex-item が flex-container 内に
収まらない時の並べ方を指定します。 デフォルトは nowrap で折り返ししない
(詰め込んで並べる)で、値を wrap にすると
折り返し(幅に収まらない分が次に並ぶ)になります。


演習 3
下記のコードを参照し、2つの flex-wrap それぞれの結果を表示させましょう。
※ flex-direction の Emmetは fxw, fxw:w 
.container {
  display: flex;
  flex-wrap: wrap; /* 折り返し */
  flex-wrap: wrap-reverse; /* 反転折り返し */
}

また、flex-flowは、flex-firection と flex-wrap を
一括で指定するプロパティがあり、以下の記述が可能です。

.container {
  display: flex;
  flex-flow: column wrap;
  /* flex-direction:column
   flex-wrap:wrap        と同じ */
}

4. FlexBox(2)

●justify-content

main-axis 方向(図の赤矢印)の整列に用いるプロパティです。
flex-item の位置や余白の取り方を指定でき、
デフォルトは、flex-start の左揃いが設定されています。


演習 1
下記のコードを参照し、各 justify-contentの結果を確認しましょう。
※ justify-contentの Emmetは jcfe、jcc、jcsb、jcsa

[CSS]

.container {
  display: flex;
  flex-flow: wrap;

  justify-content: flex-end;
  justify-content: center;
  justify-content: space-between;
  justify-content: space-around;
}





●align-items

cross axis 方向(図の青矢印)の整列に用いるプロパティです。
デフォルトでは、flex-start の上揃いが設定されています。


演習 2
下記のコードを参照し、各 justify-contentの結果を確認しましょう。
※ align-itemsの Emmetは aife、aic、ais

[CSS]

.container {
  display: flex;
  flex-flow: wrap;

  align-items: flex-end;
  align-items: center;
  align-items: stretch;
  /* .boxの height 60pxをコメントアウトする*/
}
